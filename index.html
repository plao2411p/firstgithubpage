Teoría de conjuntos

Operaciones entre conjuntos:

  • Unión -> un
  • Intersección -> in
  • Diferencia -> di
  • Complemento -> UdiA
  • Diferencia simétrica -> ds

Para realizar varias operaciones debes usar parentesis.

Ejemplos:

  • (AunB)diC
  • Ain(CdsB)
  • ((AunB)in(CinB))un(AdiB)

Conjuntos:

  • U = [1 2 3 4 5 6 7 8 9]
  • A = [1 2 3 4]
  • B = [2 4 6 8]
  • C = [3 4 5 6]
